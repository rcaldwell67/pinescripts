# playwright-mcp

This directory will contain Playwright end-to-end tests and MCP (Model Context Protocol) integration for the dashboard.

## Structure
- Place Playwright test scripts here (e.g., dashboard.spec.ts)
- Add MCP-related helpers or configuration as needed

## Setup
1. Install Playwright:
   npm install --save-dev playwright
2. (Optional) Add MCP helpers or config files for Playwright integration.

## Usage
- Run Playwright tests with:
  npx playwright test

- Add MCP integration as needed for your workflow.
